Connected to the server.
  1. Open the Terminal.
  2. Direct to the active-doc-client directory.
  3. Run command npm run start to start the application.
  4. The npm prints the address in which the tool is served in form of: http://localhost:<PORT>
  5. Open the URL in Chrome.
If the tool is running slow:
  1. Stop the application using ctrl+C.
  2. Change the port number in package.json in active-doc-client directory.
  3. Restart the app using npm run start.
  4. Open the new given URL in Chrome.
